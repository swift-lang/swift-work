
= Aurora Performance Study

== Quick start

----
$ ./run <PROCS> <PPN> <OUTPUT>
----

`OUTPUT` is the top-level of an output directory tree.

This reports a `TURBINE_OUTPUT` directory `D` .

Get results:

----
$ ./report $D
----
